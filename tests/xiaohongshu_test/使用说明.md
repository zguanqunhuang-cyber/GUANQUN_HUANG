# 小红书视频下载工具使用说明

## 📦 安装依赖

```bash
# 安装基础依赖
pip install -r requirements.txt

# 或者只安装必需的包
pip install requests you-get
```

## 🚀 使用方法

### 方法 1: 使用 Python 脚本（推荐）

```bash
python download_xiaohongshu_videos.py
```

这个脚本会：
1. ✅ 下载所有视频的封面图
2. ✅ 创建一个 shell 脚本用于批量下载视频
3. ℹ️  显示如何使用 you-get 下载视频

### 方法 2: 使用生成的 Shell 脚本

```bash
# 运行脚本会自动下载所有视频
./download_videos.sh
```

### 方法 3: 手动使用 you-get 下载

```bash
# 下载单个视频
you-get https://www.xiaohongshu.com/explore/64952ac1000000001203ca5b

# 指定输出目录
you-get -o xiaohongshu_downloads/videos https://www.xiaohongshu.com/explore/64952ac1000000001203ca5b

# 查看视频信息（不下载）
you-get -i https://www.xiaohongshu.com/explore/64952ac1000000001203ca5b
```

### 方法 4: 使用 yt-dlp 下载

```bash
# 安装 yt-dlp
pip install yt-dlp

# 下载视频
yt-dlp https://www.xiaohongshu.com/explore/64952ac1000000001203ca5b

# 下载最佳质量
yt-dlp -f best https://www.xiaohongshu.com/explore/64952ac1000000001203ca5b
```

## 🛠️ 常见问题

### Q1: you-get 下载失败怎么办？

尝试使用 yt-dlp 作为替代：
```bash
pip install yt-dlp
yt-dlp <视频链接>
```

### Q2: 如何批量下载所有视频？

编辑 Python 脚本，取消注释这几行：
```python
# video_urls = [v['url'] for v in videos]
# download_with_you_get(video_urls)
```

然后运行：
```bash
python download_xiaohongshu_videos.py
```

### Q3: 下载速度很慢怎么办？

可以尝试：
1. 使用代理
2. 使用多线程下载工具
3. 在网络状况好的时候下载

### Q4: 视频下载后没有声音？

某些视频可能需要合并音视频流，使用 ffmpeg：
```bash
# yt-dlp 会自动使用 ffmpeg 合并
yt-dlp -f best https://www.xiaohongshu.com/explore/...
```

## 🔧 高级用法

### 自定义下载配置

编辑 `download_xiaohongshu_videos.py`，修改 `XiaohongshuDownloader` 类：

```python
# 修改输出目录
downloader = XiaohongshuDownloader(output_dir="my_videos")

# 自定义请求头
downloader.session.headers.update({
    'User-Agent': '你的User-Agent',
})
```

### 下载指定质量的视频

使用 you-get:
```bash
# 列出所有可用格式
you-get -i <视频链接>

# 下载指定格式
you-get --format=<format_id> <视频链接>
```

使用 yt-dlp:
```bash
# 列出所有格式
yt-dlp -F <视频链接>

# 下载指定格式
yt-dlp -f <format_id> <视频链接>
```
